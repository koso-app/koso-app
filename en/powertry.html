PowerTry is a powerful and affordable vehicle performance measurement tool that uses high-performance satellite positioning chips combined with low-energy Bluetooth transmission for data exchange and control with mobile phones. PowerTry provides high-precision 18Hz satellite positioning data, and through data filtering and estimation algorithms, reduces the measurement time error of straight-line acceleration to within 100 milliseconds, making it a convenient performance measurement tool. K Race and K Lap apps provide systematic data analysis and display.

K RACE

Vehicle Performance Tuning Tool

K Race is a data monitoring tool that helps drivers of cars and motorcycles adjust their vehicles. By connecting to the KOSO Powertry device via Bluetooth, it can efficiently monitor vehicle data. Because the KOSO Powertry device has a highly efficient GPS and gyroscope module built in, it can record data at a high frequency interval, providing very precise and effective values. Compared to other apps that only use phone GPS data, its data is much more accurate.
  • The straight line acceleration mode is used to observe the performance of the vehicle during the acceleration process as reflected by changes in parameters.
  • The horsepower machine mode provides a simple measurement tool that can be used to understand changes in wheel horsepower during acceleration.
  • The performance calculator provides a simple calculation tool that can be used to understand the theoretical limits of vehicle performance through basic numerical input.

K LAP

Support Lap Timing & Custom Tracks

K LAP supports two types of timing races: lap timing and start/finish timing. In lap timing races, the vehicle's lap time and distance will be automatically calculated when it passes the starting point, and the results of each lap can be recorded. In start/finish timing races, the timer will start when the vehicle passes the starting point and stop when it reaches the set finish line to calculate the results of the race.
